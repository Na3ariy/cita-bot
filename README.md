# 🤖 Cita ICP Bot — Моніторинг записів для Tarjeta Conflicto Ucrania

Telegram-бот, який автоматично перевіряє наявність вільних записів на сайті  
**icp.administracionelectronica.gob.es** для процедури:  
**POLICÍA – TARJETA CONFLICTO UCRANIA** (Barcelona)

---

## ⚙️ Як запустити

### Крок 1 — Створи Telegram-бота

1. Відкрий [@BotFather](https://t.me/BotFather) у Telegram
2. Надішли `/newbot`
3. Придумай назву та username бота
4. Скопіюй **токен** (виглядає як `123456:ABCdef...`)

---

### Крок 2 — Завантаж код на GitHub

1. Створи акаунт на [github.com](https://github.com) (якщо немає)
2. Натисни **New repository** → назви `cita-bot` → **Create**
3. Завантаж всі файли цього проєкту в репозиторій

---

### Крок 3 — Задеплой на Render (безкоштовно)

1. Зайди на [render.com](https://render.com) → **Sign up with GitHub**
2. Натисни **New +** → **Web Service** → **Connect GitHub repo** → обери `cita-bot`
3. Налаштування:
   - **Type:** `Background Worker`
   - **Runtime:** `Docker`
   - **Branch:** `main`
4. У розділі **Environment Variables** додай:

   | Key | Value |
   |-----|-------|
   | `TELEGRAM_TOKEN` | токен з BotFather |
   | `CHECK_INTERVAL` | `300` (5 хв) або `180` (3 хв) |

5. Натисни **Create Background Worker**
6. Render сам збере Docker-образ і запустить бота (~3-5 хв)

---

### Крок 4 — Запусти бота в Telegram

1. Знайди свого бота за username
2. Надішли `/start`
3. Введи NIE або номер паспорта (наприклад: `X1234567Z`)
4. Введи прізвище та ім'я як у документі (наприклад: `IVANOVA OLENA`)
5. Бот підтвердить і почне моніторинг 🔔

## 📋 Команди бота

| Команда | Дія |
|---------|-----|
| `/start` | Реєстрація / зміна даних |
| `/check` | Перевірити прямо зараз |
| `/stop` | Зупинити моніторинг |
| `/status` | Переглянути статус |
| `/help` | Довідка |

## 🔧 Локальний запуск (для тестування)

```bash
# 1. Встанови залежності
pip install -r requirements.txt
playwright install chromium

# 2. Встанови змінну середовища
export TELEGRAM_TOKEN="твій_токен"

# 3. Запусти бота
python main.py

# 4. Або тільки перевір сайт (без Telegram)
python checker.py X1234567Z "IVANOVA OLENA"
```

## ❓ Часті питання

**Бот зупинився після того як знайшов запис — це нормально?**  
Так! Щоб не спамити, після знаходження слоту моніторинг автоматично зупиняється.  
Введи `/start` щоб відновити.

**Render безкоштовний — є обмеження?**  
Background Worker на Render — 750 годин/місяць безкоштовно = вистачає на 24/7.

**Що робити якщо бот помиляється (пише про помилку)?**  
Введи `/check` — бот надішле скріншот сторінки щоб зрозуміти що відбувається.  
Можливо сайт ICP тимчасово недоступний — це буває.

**Чи можна користуватись кільком людям?**  
Так! Кожен хто напише боту `/start` і введе свої дані буде окремо моніторитись.

## ⚠️ Дисклеймер

Бот призначений **тільки для моніторингу** наявності записів.  
Він **не записує автоматично** — тільки сповіщає коли з'являються слоти.  
Фінальний запис ти робиш вручну на сайті.
